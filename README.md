## Installation
```
git clone https://github.com/n1EzeR/kaspi_parser/
```
## Parser usage
```
pipenv install
pipenv shell
cd code
scrapy *command*
```

## Development
1. Use project's pipenv
2. Create new branch (e.g. feature_scrapy_product_details)
3. Switch to the new branch
4. Work (Regularly use cmd + alt + L)
5. Create Pull request
